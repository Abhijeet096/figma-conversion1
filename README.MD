# Startup 3 Landing Page Clone

## Overview

This project is a frontend clone of the Startup 3 landing page designed in Figma. The goal of this project was to improve HTML and CSS layout skills by recreating a real-world design from scratch.

## Features

* Responsive navigation bar
* Hero section with heading and description
* Call-to-action button
* Carousel indicator dots (UI only)
* Left and right navigation arrows (UI only)
* Font Awesome icons
* Flexbox-based layout

## Technologies Used

* HTML5
* CSS3
* Font Awesome

## Learning Outcomes

Through this project, I practiced:

* Semantic HTML structure
* Flexbox layout techniques
* Positioning with CSS
* Navigation bar creation
* Button styling
* Hero section design
* Converting Figma designs into code
* Debugging layout issues
